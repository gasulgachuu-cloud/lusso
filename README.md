# Lusso Chicken - Saudi Authentic Broasted Chicken

Professional restaurant website with WhatsApp ordering integration.

## ✨ Features

- 🍗 Professional restaurant branding
- 🔴 Warm, appetizing color scheme
- 📱 Mobile-responsive design
- 🔗 Direct WhatsApp ordering link
- ⏰ Hours & location display
- 📧 Contact form
- 🚀 Free lifetime hosting on GitHub Pages

## 🚀 Deploy to GitHub Pages (3 STEPS)

### Step 1: Create GitHub Account
- Go to github.com
- Sign up and verify email

### Step 2: Create Repository
- Click "New Repository"
- Name it: `yourusername.github.io` (replace with your actual username)
- Click "Create Repository"

### Step 3: Upload Files
1. Click "Add file" → "Upload files"
2. Upload all files from this folder:
   - index.html
   - css/styles.css
   - js/script.js
   - img/ folder
3. Click "Commit changes"

### ✅ DONE!
Your website is LIVE at: `https://yourusername.github.io`

Works on phone, tablet, desktop - FOREVER!

## 🎨 Customization

**Change Colors** - Edit `css/styles.css`:
- Red: #d32f2f
- Orange: #ff6f00
- Background: #ffcc99

**Change Content** - Edit `index.html`:
- Restaurant name, hours, location, pricing

**Add Images** - Place in `img/` folder and reference in HTML

### Change Colors
Edit the color codes in `css/styles.css`:
- Update color values in hexadecimal format (e.g., `#3498db`)

### Add Sections
1. Add a new `<section>` in `index.html`
2. Add navigation link in the navbar
3. Add corresponding CSS in `styles.css`

### Add Images
1. Place images in the `img/` folder
2. Reference them in HTML: `<img src="img/image-name.jpg" alt="Description">`
